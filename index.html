Courses

Intensive Introduction to Python

Our Intensive Introduction to Python is designed to help participants quickly develop the core skills necessary to apply Python to real-world problems. This course has been taught for teams of test automation engineers, data scientists, computer security experts, business analysts, optical engineers, design physicists, as well as for teams of programmers with C, C++, and Java backgrounds. At the conclusion of this course, participants should be able to understand and contribute to medium-sized Python codebases, as well as author reporting and analysis scripts, author libraries to solve common problems, and author scripts to automate their own workflows.

Advanced Applications of Python

Our advanced applications of Python is designed to help participants master foundational skills necessary to maintain medium- to large-scale Python systems. At the conclusion of this course, participants should have an understanding of how to build robust systems that use sophisticated programming techniques available in Python. Participants should understand how to write Python that may have hard performance requirements, that may belong to codebases that are organizationally complex, and that may require deep integration into other tools or systems.

Data Analysis and Numerical Computing With Python

In this two-day course, students will learn how to effectively use Python for scientific and numerical computing. This course will cover standard tools in the NumFOCUS/PyData stack, such as `numpy`, `scipy`, and `pandas`. Over the course sessions, students will work through hands-on exercises that will help them develop intuition toward best practices with Python for scientific computing.

Mastering Idiomatic Python

In this two-day course, students will learn how to write robust, idiomatic Python to solve real-world problems. Students will work through a set of refactoring exercises to identify common issues in Python programs and gain hands-on experience building well-structured, production-grade Python systems.

About Us

James Powell is a professional Python programmer and enthusiast. He started working with Python in the finance industry building reporting and analysis systems for prop trading front offices. He currently works as a consultant building data engineering and scientific computing platforms for a wide range of clients using cutting-edge open source tools like Python and React. He is a lead technical consultant for the Telecom Infa Project, which is bringing industry-standard software engineering practices to the fields of optical and backbone engineering.

He also currently serves as a Vice President at NumFOCUS, the 501(c)3 non-profit that supports all the major tools in the Python data analysis ecosystem (incl. pandas, numpy, jupyter, matplotlib, and others). At NumFOCUS, he helps build global open source communities for data scientists, data engineers, and business analysts. He helps NumFOCUS run the PyData conference series and has sat on speaker selection and organizing committees for eighteen conferences. James is also a prolific speaker: since 2013, he has given over seventy conference talks at over fifty Python events worldwide.
